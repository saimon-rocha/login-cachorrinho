
# Login com Cachorrinho
Este projeto é bem básico utilizando somente HTML, CSS e Javascript. Ao digitar a senha o cachorrinho deve ficar com os olhos tapados e quando clicar para ver a senha o cachorrinho deve destapar apenas um olhinho e ao voltar a digitar a senha ele cobre todos os olhos novamente.

## Funcionalidades Implementadas
Apenas faça o download e abra o arquivo html com o navegador do google chrome
## Stack utilizada

**HTML**
**CSS**
**JAVASCRIPT**
